# 🍽️ Aura Premium Luxury Restaurant - Frontend Website

A modern, responsive, and visually appealing **Restaurant Frontend Website** built using HTML, CSS, and JavaScript.

This project showcases a premium restaurant landing page with a clean UI, responsive layout, smooth interactions, and an SEO-friendly structure. It was created as part of my web development learning journey and serves as a frontend portfolio project.
